## Project Stack

* **Frontend Engine:** React (Vite)
* **Styling Tools:** Tailwind CSS
* **Animations:** Framer Motion (Scroll parallax, hover elasticity)
* **Messaging:** EmailJS (Browser-side environment)
* **Assets handling:** Lucide React

## Local Installation

To run this portfolio template on your local machine:

1. Clone this repository down locally.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the fast local development server:
   ```bash
   npm run dev
   ```
4. Access `http://localhost:5173/` in your browser.
